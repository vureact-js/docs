# 安装

## 1. 环境要求

本库依赖以下版本的环境，确保你的项目满足以下条件后再进行安装：

- React：`18.2.0` 及以上版本
- Node.js：`20.0.0` 及以上版本

## 2. 安装方式

支持 npm、pnpm、yarn 三种主流包管理工具，你可以根据项目习惯选择对应的安装命令：

### npm

```bash
npm install react-vue3-components
```

### pnpm

```bash
pnpm add react-vue3-components
```

### yarn

```bash
yarn add react-vue3-components
```
