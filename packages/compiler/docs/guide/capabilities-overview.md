# 能力矩阵总览

本页给出实验版当前能力边界，用于评估项目是否适配 VuReact。

## 判定口径

- `支持`：主路径稳定可用
- `部分支持`：可用，但有约束、降级或额外改造
- `不支持`：当前无法保证正确转换

## 模块级矩阵

| 领域        | 状态     | 你需要关注什么                                       |
| ----------- | -------- | ---------------------------------------------------- |
| Template    | 支持     | 自定义指令受限、复杂动态表达式可能告警或降级         |
| Script      | 支持     | 宏/API 适配可用，但 Hook 顶层规则非常严格            |
| Style       | 支持     | 单 style + scoped/module 可用，多 style/cssVars 受限 |
| Router 相关 | 部分支持 | 语法和API有适配，工程入口与部分路由组织仍需手动改    |

## 场景适配矩阵

| 场景                    | 适配度 | 建议                  |
| ----------------------- | ------ | --------------------- |
| 新项目按约定开发        | 高     | 推荐优先采用          |
| 渐进式迁移（分模块）    | 中     | 先试点、再扩圈        |
| 复杂旧项目一次性迁移    | 低     | 不建议作为首选方案    |
| 不使用 `<script setup>` | 低     | 建议使用现代 Vue 语法 |
| 对 Vue 2 使用           | 无     | 完全不建议            |
